# Project Documentation Index

## Overview

This folder contains step-by-step documentation of the Olist End-to-End Data Project.

Each file represents a specific stage of the data pipeline, from raw data ingestion to analytics and machine learning.

---

## Documentation Structure

### 01_project_scope.md
Defines project objectives, scope, and expected outputs.

### 02_database_creation.md
Explains how OlistDB (operational database) was created and populated.

### 03_star_schema_design.md
Describes the design of OlistDW and the Star Schema model.

### 04_data_cleaning.md
Details data quality issues and cleaning processes.

### 05_eda.md
Contains exploratory data analysis and key insights.

### 06_dashboard_design.md
Explains Power BI dashboard structure and KPIs.

### 07_machine_learning.md
Covers ML models, features, and evaluation.

### 08_etl_improvements.md
Documents improvements made to ETL logic and fact/dimension modeling.

### 09_data_dictionary.md
Defines all tables and columns in OlistDB and OlistDW.

---

## How to Use This Folder

Start from `01_project_scope.md` and follow sequentially to understand the full project lifecycle.

Each file is designed to be **self-contained and readable independently**.
